# Kahvila Lumi - ravintolasivun demo

Tämä on moderni kahvila / ravintola -nettisivu, joka on tehty HTML:llä, CSS:llä ja JavaScriptillä.

## Tiedostot

```text
index.html
style.css
script.js
README.md
```

## Mitä sivulla on?

- Moderni navigaatio
- Iso hero-osio
- Menu-kortit JavaScriptillä
- Tarina-osio
- Asiakasarviot
- Kuvagalleria
- Yhteystiedot ja aukioloajat
- Responsiivinen mobiiliversio

## Käynnistys

Avaa `index.html` selaimessa.

Voit julkaista sivun GitHub Pagesilla:

1. Mene repositoryn asetuksiin
2. Avaa Pages-osio
3. Valitse branchiksi `main`
4. Valitse kansioksi `/root`
5. Tallenna

## Huomio

Sivulla käytetään ulkoisia kuvia Unsplashista. Ne voi myöhemmin vaihtaa yrityksen omiin kuviin.
